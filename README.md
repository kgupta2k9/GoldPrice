# Gold Price Tracking Project

## Overview
This project tracks the price of gold over time and provides insights into trends and patterns.

## Features
- Real-time gold price data
- Historical price charts
- Notifications for price changes

## Getting Started
### Prerequisites
- Python 3.x
- Required libraries (see `requirements.txt`)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kgupta2k9/GoldPrice.git
   ```
2. Navigate to the project directory:
   ```bash
   cd GoldPrice
   ```
3. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the main script:
```bash
python main.py
```
